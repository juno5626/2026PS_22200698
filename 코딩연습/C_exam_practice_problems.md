# C언어 코딩 시험 대비 문제 세트

이 문제 세트는 교수님이 사용하신 코드 스타일을 기반으로\
**반복문 → 배열 → 함수 → 포인터 → 구조체 → 파일 → malloc** 순서로
난이도가 올라가도록 구성되었습니다.

총 **18문제**이며 시험 대비용으로 설계되었습니다.

------------------------------------------------------------------------

# Level 1 --- 기초

## P01

bmi = 몸무게(kg) ÷ (키(m) × 키(m))
사용자로부터 **키(cm)와 몸무게(kg)** 를 입력받아 **BMI를 계산하여
출력**하는 프로그램을 작성하시오.

------------------------------------------------------------------------

## P02

BMI 값을 계산한 뒤 다음 기준에 따라 **비만 여부를 출력**하시오.

-   BMI ≥ 25 → 비만
-   BMI \< 25 → 정상

------------------------------------------------------------------------

## P03

5명의 키와 몸무게를 입력받아 **각 사람의 BMI를 출력**하시오.

조건\
- 반복문 사용

------------------------------------------------------------------------

## P04

5명의 BMI 중 **비만(BMI ≥ 25)인 사람 수**를 출력하시오.

------------------------------------------------------------------------

## P05

5명의 BMI 중 **가장 높은 BMI 값을 출력**하시오.

------------------------------------------------------------------------

## P06

5명의 **평균 BMI**를 계산하여 출력하시오.
